I am a computer engineering student from Nepal. I usually start projects before I
fully understand the problem, then learn what I need while building them.

Most of my time currently goes into a major project about communication between
reinforcement-learning agents. My first experiment trained agents that traded
better than random, but it did **not** prove their messages were useful. That
result sent me back to redesign the environment and run better experiments.
