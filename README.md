## About me

I am a computer engineering student from Nepal who learns by building. I am most
interested in intelligent systems, backend security, and software that solves
small but real problems.

My main project studies communication between reinforcement-learning agents. The
first trained agents traded better than random, but their messages did not make a
meaningful difference. Instead of presenting that as a success, I redesigned the
environment and started a better experiment.

## Selected work

### Emergent Trading Language

> Can autonomous agents develop a communication system that actually helps them
> cooperate?

My major project places two agents in a trading environment where they exchange
short messages before proposing trades. I am comparing normal, removed, and
randomized messages to test whether communication itself improves performance.

**Built with:** Python, PyTorch, policy gradients, experiment analysis<br>
**Current result:** agents learned to trade, but useful language is not proven yet
